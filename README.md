# VECORDIA

**VECORDIA** — это аркадный хоррор с элементами рогалика, разработанный в качестве учебного проекта (MVP-билд). Основная задача игрока — погрузиться в мрачную атмосферу и успеть завершить все ритуалы за ограниченное время.

---

##  Игровой процесс и концепция

* **Жанр:** Аркадный хоррор от первого лица.
* **Главная цель:** Успеть провести все необходимые ритуалы на локации до того, как истечет таймер и уровень тревоги достигнит 100%.
* **Время на раунд:** ровно **4 минуты**.
* **Атмосфера:** Напряженный тайм-менеджмент в условиях хоррор-окружения, где каждая секунда на счету.

---

##  Технический стек 

* **Игровой движок:** Unity
* **Платформа:** Windows (сборка и тестирование проводились исключительно на этой ОС).
* **Система контроля версий:** GitHub.



---

##  Текущий статус проекта (MVP)

На текущий момент проект представляет собой играбельный MVP-билд. В нем реализовано:
* Полностью готовая и настроенная игровая локация.
* Базовые механики перемещения и взаимодействия с объектами.
* Логика проведения 4-х ритуалов и система глобального таймера на 4 минуты.
* Скрытая шкала тревоги

---

##  Как запустить игру (Быстрый старт)

Для того чтобы протестировать текущую версию игры, не обязательно устанавливать движок Unity. Достаточно запустить скомпилированный билд:

1. Перейдите по ссылке https://drive.google.com/drive/folders/17LfOeP_nokASagdnNBwTLu9sgISgb975?usp=sharing
2. Скачайте архив с самым актуальным релизным билдом под Windows.
3. Распакуйте архив в любую удобную папку на вашем компьютере.
4. Запустите исполняемый файл `.exe` игры.


## Разработка (исходный код)

Если вы хотите скачать исходный код, принять участие в разработке или просто запустить проект в редакторе:

1. **Скачайте и установите Git LFS** (нужно для 3D-моделей) с сайта [git-lfs.github.com](https://git-lfs.github.com/) и введите в консоли команду:
```bash
git lfs install

```


2. **Скачайте сам проект** (можете через консоль, можете через приложение **GitHub Desktop**):
```bash
git clone https://github.com/youtube-samsung/VECORDIA.git

```


3. **Откройте в Unity:** Просто добавьте скачанную папку через Unity Hub.

*(Если моделей нет , введите команду: `git lfs pull`)*
